# VISILETCA
Consol VISILETCA game in PY

Классическая игра "Виселица", написанная на Python, где игрок угадывает слово по буквам.

## Описание игры

Игрок пытается угадать загаданное слово, предлагая буквы. За каждую неправильную букву добавляется элемент виселицы. Цель - угадать слово до того, как будет нарисована полная виселица.

##  Особенности

- **3 уровня сложности**: легкий, средний, сложный
- **Система подсчета очков**: учитывает длину слова, оставшиеся попытки и сложность
- **Визуализация**: ASCII-графика виселицы
- **Сохранение результатов**: статистика игр записывается в файл
- **Русскоязычный интерфейс**
